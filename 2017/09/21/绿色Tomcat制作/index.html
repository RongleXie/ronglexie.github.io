绿色版Tomcat制作
=

一、setclasspath.bat文件添加
-
rem set user jdk and jre home
set JAVA_HOME=..\jdk1.7.0_13_32
set JRE_HOME=..\jdk1.7.0_13_32\jre

二、新建startup.bat
-
set “CURRENT_DIR=%~dp0”
set “CATALINA_HOME=%CURRENT_DIR%\tomcat7”
set JRE_HOME=%CURRENT_DIR%\JRE
set CLASSPATH=%JRE_HOME%\lib
set PATH=%PATH%;%JRE_HOME%\bin
set JAVA=java
call %CATALINA_HOME%\bin\startup.bat
